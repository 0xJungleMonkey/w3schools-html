Js Math Objects

Math.round(x) returns rounded integer

Math.pow(x,y) returns The Y power of X

Math.sqrt(x) returns the square root of X

Math.abs(x) returns the absolute value of X

Math.ceil(x) returns a number's upward-rounded value

Math.floor(x) returns a number's round-down value

Math.sin(x) Math.cos(x)

Math.min(x,y,z,w,q) Math.max(x,w,e,r,t,q)

Math.random() return random numbers between 0 and 1
