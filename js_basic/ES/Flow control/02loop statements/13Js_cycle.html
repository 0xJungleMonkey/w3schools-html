Loop statements

1. JS FOR Loop

for 循环呆板语法
Grammar:
for ( i = 0; the range of i; i++) {
statement; }
注意：②位置如果为空，或者②条件设置的不合理， 也会出现死循环

for 内部可以嵌套 if 语句,嵌套 for 循环

for/in 语句循环遍历对象的属性。

for (variable in object){ statement }

}while (条件);

3. While Loop
